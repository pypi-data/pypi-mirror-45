はじめに¶

このパッケージは，spAudio ライブラリ のpython版で，フルデュプレックスのオーディオデバイス入出力を可能とする spaudio モジュールと，WAV，AIFF，MP3，Ogg Vorbis，FLAC，ALAC，rawなど，様々なファイル形式の読み書きをプラグインにより可能とする spplugin モジュールが用意されています．

インストール¶

バイナリーパッケージをインストールする際には， pip コマンドを使うことができます．

pip install spaudio

もし， Anaconda または Miniconda を使用している場合は，"bannohideki" チャンネルを指定して conda コマンドを実行してください:

conda install -c bannohideki spaudio

なお，このパッケージはPython 2はサポートしていないのでご注意ください．

Linuxバージョンは，spPlugin のインストール（特に，オーディオデバイス入出力には， PulseAudio を利用した pulsesimple プラグインが必要になります）も必要になります．これは， dpkg (Ubuntu) または rpm (CentOS) のコマンドを，以下のいずれかのファイルと共に使用してインストールできます．

    • i386: http://www-ie.meijo-u.ac.jp/labs/rj001/archive/deb/ubuntu18/spplugin_0.8.5-4_i386.deb
  • Ubuntu 16

  • Ubuntu 14

    • i386: http://www-ie.meijo-u.ac.jp/labs/rj001/archive/deb/ubuntu14/spplugin_0.8.5-4_i386.deb
