  Když linkaOn == 1

  Pokud shuttleXXAtStationXY == 1 && replenishStoreR3 == 1
  Když  storeR3Count == 0
  Pak   robotR1ProgramNumber == 29 && edge(robotR1ProgamStart, 1)

Scénář: Doplnění skladu kostiček - Robot R1 - Doplnění zásob ve skladu kostiček z vozíčku - kostičky 4x2
